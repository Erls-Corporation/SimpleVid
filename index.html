jQuery Plugin for Fluid Video

In general, it is best to use a service (YouTube, Vimeo, etc.) for hosting video, but sometimes this isn’t an option. SimpleVid is a free and easy way to host and embed your own fluid videos. Uses flash to cover browsers that don’t support h.264, so you can encode once (as a baseline h.264 mp4) and play anywhere.

How to Use

Add video element to your html in a container div that scales the way you like.

<div id="container">
    <video id="myVideo" width="640" height="360" poster="img/poster.jpg" controls>
        <source src="vid/video.mp4" type="video/mp4"  />
    </video>
</div>

First, link to jQuery and then simplevid.js. Next, initialize the plugin and apply any custom settings to it. Last, use the plugin to target the video element(s) on the page.

<script src="http://code.jquery.com/jquery-1.7.min.js"></script>
<script src="js/simplevid.js"></script>
<script>
    var simplevid = new $.simplevid({
        player: 'swf/vidobject.swf',
        buttonColor: '#FFFFFF',
        accentColor: '#BBBBBB',
        backgroundColor: '#111111',
        scaleVideo: true,
        fullScreen: true
    });
    simplevid.embed("myVideo");
</script>

Parameters are:

  • player: path to vidobject.swf
  • buttonColor: controls button color (default: #ffffff)
  • accentColor: controls accent color (default #bbbbbb)
  • backgroundColor: controls background color (default: #111111)
  • scaleVideo: set to false to disable video scaling
  • fullScreen: fullScreen: set to false to disable full screen mode
